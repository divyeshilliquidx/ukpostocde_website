Authentication

Secure your API requests with bearer token authentication.

curl -H "Authorization: Bearer YOUR_API_KEY" \
     -H "Content-Type: application/json" \
     https://api.dataflow.com/v1/data

Base URL: https://api.dataflow.com/v1

API Endpoints

GET /data/{id}

Retrieve specific data by ID

GET /data/12345

POST /data

Create new data entry

POST /data
{
  "name": "Sample Data",
  "value": 42
}

Code Examples

JavaScript

const response = await fetch('https://api.dataflow.com/v1/data', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
console.log(data);

Python

import requests

headers = {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
}

response = requests.get('https://api.dataflow.com/v1/data', headers=headers)
data = response.json()
