Authentication

Secure your API requests with bearer token authentication.

curl -H "Authorization: Bearer YOUR_API_KEY" \
     -H "Content-Type: application/json" \
     https://api.dataflow.com/v1/data

Base URL: https://api.dataflow.com/v1

Getting Your API Key

To get started, you'll need to obtain an API key from your dashboard:

  1. Sign up for a DataFlow account
  2. Navigate to the API Keys section
  3. Generate a new API key
  4. Copy and securely store your key

API Endpoints

GET /data/{id}

Retrieve specific data by ID

GET /data/12345

Response:

{
  "id": "12345",
  "name": "Sample Data",
  "value": 42,
  "created_at": "2024-01-15T10:30:00Z"
}

POST /data

Create new data entry

POST /data
{
  "name": "Sample Data",
  "value": 42
}

PUT /data/{id}

Update existing data entry

PUT /data/12345
{
  "name": "Updated Data",
  "value": 84
}

DELETE /data/{id}

Delete data entry

DELETE /data/12345

Code Examples

JavaScript

const response = await fetch('https://api.dataflow.com/v1/data', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
console.log(data);

Python

import requests

headers = {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
}

response = requests.get('https://api.dataflow.com/v1/data', headers=headers)
data = response.json()

Node.js

const axios = require('axios');

const config = {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
};

axios.get('https://api.dataflow.com/v1/data', config)
  .then(response => console.log(response.data))
  .catch(error => console.error(error));

Rate Limit Headers

Every API response includes rate limit information:

X-RateLimit-Limit: 1000
X-RateLimit-Remaining: 999
X-RateLimit-Reset: 1640995200

Rate Limit Exceeded

When you exceed your rate limit, you'll receive a 429 status code:

{
  "error": "Rate limit exceeded",
  "message": "Too many requests. Please try again later.",
  "retry_after": 3600
}
