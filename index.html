Getting Started

The UK Postcode API provides instant access to comprehensive UK geographic data. Get started in minutes with our RESTful API.

curl -X GET "https://api.ukpostcode.com/v1/postcodes/SW1A1AA" \
  -H "Authorization: Bearer YOUR_API_KEY"

Base URL: https://api.ukpostcode.com/v1

Authentication

All API requests require authentication using an API key in the Authorization header.

Authorization: Bearer YOUR_API_KEY

API Endpoints

GET /postcodes/{postcode}

Lookup detailed information for a specific postcode

GET /postcodes/SW1A1AA

POST /postcodes

Bulk lookup multiple postcodes in a single request

POST /postcodes

GET /postcodes/{postcode}/validate

Validate if a postcode exists and is correctly formatted

GET /postcodes/SW1A1AA/validate

Code Examples

JavaScript

const response = await fetch('https://api.ukpostcode.com/v1/postcodes/SW1A1AA', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  }
});
const data = await response.json();

Python

import requests

headers = {'Authorization': 'Bearer YOUR_API_KEY'}
response = requests.get('https://api.ukpostcode.com/v1/postcodes/SW1A1AA', headers=headers)
data = response.json()
